# HypertensionGuard AI

AI-powered Blood Pressure Prediction System using Machine Learning.

## Features
- Real-time hypertension risk prediction
- Medical recommendations
- Interactive web interface

## How to Run
1. Install Python 3.8+
2. Install dependencies: `pip install flask scikit-learn joblib numpy pandas`
3. Run: `python app.py`
4. Open browser: `http://127.0.0.1:5000/`

## Tech Stack
- Python, Flask
- Scikit-learn (ML)
- HTML/CSS
